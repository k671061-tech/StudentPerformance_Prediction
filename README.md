# 🎓 Student Performance Prediction using Machine Learning

## 📌 Overview

This project uses Machine Learning to predict a student's academic performance based on different factors such as study hours, attendance, previous exam scores, assignments, and other academic details. The model helps identify students who may need additional support to improve their performance.

---

## 🎯 Objectives

- Predict student academic performance.
- Analyze factors affecting student results.
- Improve educational decision-making.
- Learn Machine Learning concepts through a real-world project.

## ▶️ How to Run

1. Install Python.
2. Install required libraries.

```bash
pip install pandas numpy matplotlib scikit-learn streamlit
```

3. Open the project in VS Code or Jupyter Notebook.
4. Run the notebook or app.
5. Enter student details.
6. Click **Predict** to view the predicted performance.
